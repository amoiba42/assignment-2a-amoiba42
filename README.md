## Controls

The current interaction controls used by the generated Version 4 scene are:

| Key | Action |
|---|---|
| `W` `A` `S` `D` | Move the player forward, left, backward, and right |
| `Q` | Move the player downward |
| `E` | Move the player upward, or interact with the cupboard when looking at it |
| `F` | Collect a book when looking at it |
| `T` | Play the trumpet when looking at it |
| `R` | Start rotating a rotatable object when looking at it; hold to rotate with mouse movement and release to stop |
| Mouse | Look around; while holding `R`, horizontal mouse movement rotates the selected object |

## Notes

- Book interaction is line-of-sight based: the player must be looking at the book and then press `F`.
- Cupboard interaction shares the `E` key with vertical movement, so looking at the cupboard takes priority over upward movement.
- The chest blasts automatically when the required conditions are met.
